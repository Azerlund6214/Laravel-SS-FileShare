# Laravel-SS-FileShare
 Файлообменник

Ссылка на рабочий сайт: https://fileshare.awc-capital.com/

![Интерфейс](https://i.imgur.com/C9Jvi0r.png)

# Функционал

1) Загрузка файлов, и генерация сокращенной ссылки для скачки
2) Настраиваемые лимиты на размер файла и общий размер хранилища
3) Файлы имеют ограниченный срок хранения
4) Автоудаление устаревших файлов
5) Удаление файлов через админку

# Инструкция по установке.

1) Подключить свою бд в файле .env
2) Выполнить миграции - "php artisan migrate"
3) Готово

# Вход в админку.

Перейти по ссылке: (localhost.com)/admin/123pass

У администратора появляется кнопка "Удалить" возле каждой записи.

Пароль меняется в коде контроллера. (в начале класса).

Предусмотрена защита - удалить запись просто зная URI экшена не получится.
